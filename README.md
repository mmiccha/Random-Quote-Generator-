# Random Quote Generator

![Python](https://img.shields.io/badge/Python-3.x-3776AB?logo=python)

Простое графическое приложение для генерации случайных цитат с возможностью фильтрации и добавления новых записей.

## Описание программы

Это графическое приложение, которое генерирует случайные цитаты из предопределённого списка. Пользователь может:
* просматривать случайные цитаты;
* фильтровать их по автору и теме;
* добавлять новые цитаты.

Вся история сгенерированных цитат сохраняется в JSON‑файл и загружается при запуске программы.

## Возможности

* Генерация случайной цитаты по нажатию кнопки.
* Просмотр истории всех сгенерированных цитат.
* Фильтрация истории по автору и теме.
* Добавление новых цитат вручную.
* Автоматическое сохранение истории в JSON‑файл (`history.json`).
* Загрузка истории при запуске программы.

## Примеры использования программы
1. Получить случайную цитату:

Нажмите кнопку «Сгенерировать цитату». Цитата появится в верхней части окна.

2. Отфильтровать историю цитат:

Введите имя автора (или его часть) в поле «Автор».

Введите тему (или её часть) в поле «Тема».

Нажмите кнопку «Применить фильтр». В списке истории отобразятся только подходящие цитаты.

3. Добавить свою цитату:

Нажмите кнопку «Добавить цитату».

Во всплывающих окнах последовательно введите:

текст цитаты;

автора цитаты;

тему цитаты.

После подтверждения новая цитата добавится в общий список и появится в истории.

4. Сбросить фильтрацию:

Нажмите кнопку «Сбросить». В списке истории снова отобразятся все сохранённые цитаты.

5. Автоматическое сохранение:

Все действия (генерация, добавление, фильтрация) не требуют ручного сохранения. 
История автоматически записывается в файл history.json после каждого изменения. 
При следующем запуске программы она будет загружена.
